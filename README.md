# Customer-Shopping-Behavioural-Analysis-
A comprehensive end-to-end data analytics project that explores and uncovers patterns in customer purchasing behaviour across retail transactions. The project involves collecting raw transactional data in a MySQL, data cleaning, transformation, and EDA using Python, and visualising actionable insights through an interactive Power BI dashboard.
# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading to delivering business insights through visualization and reporting. The goal is to analyze the dataset, uncover trends, and present actionable insights using industry-standard tools.

---

## 📂 Dataset

* The dataset contains structured data relevant to business operations (e.g., sales, customers, or transactions).
* It includes multiple attributes such as numerical, categorical, and date fields.
* Data quality issues such as missing values, duplicates, and inconsistencies were handled during preprocessing.

---

## 🛠️ Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **Pandas, NumPy, Matplotlib, Seaborn** – Data manipulation and visualization
* **MySQL** – Data querying and transformation using SQL
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation
* **Excel (optional)** – Data validation and quick checks

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas.
* Performed initial inspection of data structure and types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions, trends, and relationships.
* Identified key patterns and anomalies using visualizations.

### 3. Data Cleaning

* Handled missing values (imputation or removal).
* Removed duplicate records.
* Standardized formats (e.g., date, categorical values).
* Ensured data consistency for accurate analysis.

### 4. SQL Analysis (MySQL)

* Loaded cleaned data into MySQL.
* Wrote SQL queries using:

  * Joins
  * Aggregations
  * Window functions
* Extracted insights such as top performers, trends, and KPIs.

### 5. Dashboard Creation (Power BI)

* Built an interactive dashboard with:

  * KPIs and summary metrics
  * Trend analysis visuals
  * Filters and slicers for user interaction

### 6. Reporting & Presentation

* Created a detailed report summarizing findings.
* Designed a professional presentation (PPT) using Gamma to communicate insights effectively.

## 🚀 How to Run

1. **Clone the repository**

   ```bash
   git clone <repository-link>
   ```

2. **Install dependencies**

   ```bash
   pip install pandas numpy matplotlib seaborn mysql-connector-python
   ```

3. **Run Python scripts**

   * Load and clean data
   * Perform EDA

4. **Set up MySQL**

   * Import dataset into MySQL
   * Run SQL scripts provided in the project

5. **Open Power BI file**

   * Load dataset
   * Explore the dashboard

   * Check the report document
   * Review the PPT created using Gamma
